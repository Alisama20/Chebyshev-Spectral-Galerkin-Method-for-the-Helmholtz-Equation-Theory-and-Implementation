# Chebyshev-Spectral-Galerkin-Method-for-the-Helmholtz-Equation-Theory-and-Implementation
Python implementation of a Chebyshev spectral–Galerkin discretization of the 2D Helmholtz equation. Includes theoretical derivation of the weak formulation, construction of basis functions satisfying Dirichlet boundary conditions, matrix assembly using Kronecker products, and exponential convergence study.
